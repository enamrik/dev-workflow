# Configuration Guide

> This guide is part of the [dev-workflow documentation](../README.md).

Guide to configuring dev-workflow: templates, hooks, types, and settings.

_Documentation coming soon - see [Issue #162](https://github.com/enamrik/dev-workflow/issues/162) for progress._

## Configuration Files

- `.track/config.json` - Project settings
- `.track/templates/issues/` - Issue templates
- `.track/templates/tasks/` - Task templates
- `.track/types.md` - Custom type definitions
- `.track/hooks/` - Task lifecycle hooks (YAML)
