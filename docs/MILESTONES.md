# Milestones

> This guide is part of the [dev-workflow documentation](../README.md).

Guide to time-bounded planning with milestones.

_Documentation coming soon - see [Issue #162](https://github.com/enamrik/dev-workflow/issues/162) for progress._

## Features

- Create milestones with start/end dates
- Automatic status computation (PLANNED, IN_PROGRESS, DELAYED, COMPLETED)
- Assign issues to milestones
- Deadline tracking
