# Claude Skills Guide

> This guide is part of the [dev-workflow documentation](../README.md).

Guide to the 7 Claude skills that enable natural language workflow automation.

_Documentation coming soon - see [Issue #162](https://github.com/enamrik/dev-workflow/issues/162) for progress._

## Available Skills

- dwf-work-request - Entry point for all new work
- dwf-manage-issue - Issue creation, update, merge, import
- dwf-plan-issue - Implementation planning with SPIKE handling
- dwf-work-task - Task dispatch decision (worker vs inline)
- dwf-worker-task - Task execution lifecycle
- dwf-configure-github - GitHub integration setup
- dwf-manage-milestone - Milestone management
